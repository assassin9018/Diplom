Для входа в приложение сначала необходимо пройти авторизацию в программе. Для этого нужно будет ввести уникальный Логин пользователя и его пароль в соответствующие поля на форме авторизации.
Если пара логин\пароль введена не правильно, приложение сообщит об этом.
В дальнейшем подразумевается, что во всех формах внесения новых и редактирования уже существующих данных, все поля являются обязательными для заполнения.
Также на всех формах, которые содержат выпадающие списки для выбора тех данных, которые могут быть общими для нескольких сотрудников, присутствуют кнопки с символом "+", которые открывают форму для добавление новой информации в эти списки.

После авторизации пользователю доступна основная рабочая форма "Отдел кадров". На этой форме пользователь может просматривать основную информацию по всем сотрудникам, фильтровать их по должностям и подразделениям, сортировать по всем полям таблицы и производить поиск по имени, фамилии и отвеству работников.
Также в зависимости от наличия прав, пользователь может через главное меню выполнять следующие действия:
	- Через подменю "Сервис":
		- Нанять нового сотрудника
		- Добавить нового пользователя и задать ему права доступа
		- Открыть форму для просмотра всех отпусков по всем сотрудникам
		- Открыть форму для просмотра всех командировок по всем сотрудникам
		- открыть форму для просмотра и редактирования всех пользователей системы
	- Через подменю "Справка":
		- Вызвать руководство пользователя
	- Через контекстное меню для любой из строк в таблице:
		- Просмотреть подробную информацию в личное карточке сотрудника
		- Отредактировать данные в личной карточке сотрудника
		- Отправить сотрудника в командировку
		- Отправить сотрудника в отпуск
		- Уволить сотрудника
		
Формы "Пользователь", "Организация", "Город", "Образование", "Тип образования", "Подразделение", "Специальность", "Должность" являются формами ввода данных и не несут в себе дополнительного функциала, за исключние вызова других форм, которые необходимо для добавление информации, от которой зависит текущая форма.
Все поля на формах ввода данных обозначены подписями, которые соответсвуют требуемым в них данным.

Форма "Личная карточка сотрудника". 
Форма служит для принятия новых сотрудников на работу, а также для просмотра подробной информации и редактирования данных уже существующих сотрудников.

Форма "Командировка".
Через данную форму можно отправить любого из работников в командировку.

Форма "Командировки".
На данной форме представлены все командировки по всем сотрудникам в виде таблицы. Пользователю доступна сортировка по всем полям таблицы, добавление новых командировок через вкладку "Сервис" в главном меню, а также удаление старых командировок через выпадающее меню в таблице.

Форма "Отпуск".
Через данную форму можно оформить отпуск для любого из работников.

Форма "Отпуска"
На данной форме представлены все отпуска по всем сотрудникам в виде таблицы. Пользователю доступна сортировка по всем полям таблицы, добавление новых отпусков через вкладку "Сервис" в главном меню, а также удаление старых отпусков через выпадающее меню в таблице.
